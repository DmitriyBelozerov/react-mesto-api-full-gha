Проект: "Место"

Цель: Каждый человек может поделиться с другими участниками проекта фотографиями своего любимого места. 

Использумые технологии: React.

В планах: Создать профили участников проекта, добавить списки друзей, разделить фотографии по пользователям и добавить сортировку фото. "Допилить" валидацию форм.
